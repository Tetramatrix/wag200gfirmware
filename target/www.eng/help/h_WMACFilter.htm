Wireless Tab / Wireless Access


Wireless access can be controlled by using the MAC addresses of the wireless devices transmitting within your network radius.

Wireless Network Access

Allow All

Select this if you wish to allow all wireless devices access, regardless of their MAC address. (Wireless stations must still use the correct security settings.)

Restrict Access

Select this if you wish to restrict wireless access to known (trusted) stations, or wish to block certain (untrusted) stations. Click Update Filter List to create or modify the list of wireless stations. Note that these stations must still use the correct security settings.

Prevent listed computers from accessing the wireless network.

Clicking this radio button will prevent wireless access by the stations in the filter list. All other stations will be allowed.

Permit only listed computers to access the wireless network.

Clicking this radio button will allow wireless access by the stations in the filter list. All other stations will be blocked.
 
Edit MAC Address Access List Click the Edit MAC Address Access List button to view the MAC Address Filter List sub-screen, which allows to create or modify the list of Wireless stations.
 

Click the Save Settings button to apply your changes or Cancel Changes button to cancel.
