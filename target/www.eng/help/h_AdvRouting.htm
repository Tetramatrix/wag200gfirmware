Setup Tab / Advanced Routing


This tab is used to set up the Router advanced functions. Operating Mode allows you to select the type(s) of advanced functions you use. Dynamic Routing (RIP) will automatically adjust how packets travel on your network. Static Routing sets up a fixed route to another network destination.

Operating Mode NAT
Select the NAT mode in which this Router will function. Normally, this should be Enabled, this allows all devices on your LAN to share the same WAN (Internet) IP address.
If Disabled, you either need another Router to act as the Internet Gateway, or all PCs on your LAN must be assigned (fixed) Internet IP addresses.
 
Dynamic Routing (RIP) This feature enables the Router to automatically adjust to physical changes in the network’s layout and exchange routing tables with the other router(s). The Router determines the network packets route based on the fewest number of hops between the source and the destination.

RIP

Select Enabled to enable RIP. If you don't have other Router on your LAN, you can leave this Disabled.

Transmit RIP Version

Select the version of Transmit RIP you wish to use. This should match the version supported by other Routers on your LAN.

Receive RIP Version

Select the version of Receive RIP you wish to use. This should match the version supported by other Routers on your LAN.

 
Static Routing Static Routing allows you define the routing path to other networks or network segments. The following settings are available.

Select set number

Select the set number (routing table entry number) that you wish to view or configure.
Click the Delete This Entry button to delete (clear) the entry if desired.

Destination LAN IP

This is the address of the remote network or host to which you want to assign a static route.

Subnet Mask

The Subnet Mask determines which portion of a Destination LAN IP address is the network portion, and which portion is the host portion.

Default Gateway

This is the IP address of the gateway device that allows for contact between the Router and the remote network or host.

Hop Count

Enter the number of intermediate connections for the Route here.

Click the Show Routing Table button to view all the Static Routes you have defined.
 


Click the Save Settings button to apply your changes or Cancel Changes button to cancel.
