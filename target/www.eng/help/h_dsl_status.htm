Status Tab / DSL Connection


The DSL Connection screen on the Status Tab displays the status of your DSL connection.

Click the Refresh button to update the data displayed on screen.

DSL Connection DSL Status
This indicates whether or not the DSL link is currently connected.

DSL Modulation Mode

This shows the DSL Modulation Mode.

DSL Path Mode

This shows the DSL Path Mode.

Downstream Rate

This shows the download speed (for traffic from the Internet to this device) of the DSL connection.

Upstream Rate

This shows the upload speed (for traffic from this device to the Internet) of the DSL connection. For ADSL connections, the upstream rate is typically 25% of the downstream rate.

Downstream Margin

This shows the Downstream Margin.

Upstream Margin

This shows the Upstream Margin.

Downstream Line Attenuation

This shows the Downstream Line Attenuation.

Upstream Line Attenuation

This shows the Upstream Line Attenuation.

Downstream Transmit Power

This shows the Downstream Transmit Power.

Upstream Transmit Power

This shows the Upstream Transmit Power.
PVC Connection

Encapsulation

This displays the Encapsulation method of the VC (e.g. RFC 1483 Routed, RFC 2516 PPPoE).

Multiplexing

The multiplexing method is either LLC or VC.

Qos

This will indicate UBR, CBR, or VBR.

Pcr Rate

This displays the current Pcr Rate. If using UBR above, this rate is determined automatically.

Scr Rate

This displays the current Scr Rate. If the Qos value is UBR or CBR, this rate is determined automatically.

Autodetect

This indicates if the VPI and VCI values below were auto detected.

VPI

The VPI value for this VC.

VCI

The VCI value for this VC.

Enable

This indicates if the selected VC is enabled.

PVC Status

The current status of the selected VC.
