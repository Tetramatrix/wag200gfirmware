Status Tab / Local Network


The Local Network screen on the Status Tab displays the status of your LAN.

Click the Refresh button to update the data displayed on screen.

Local Network MAC Address
This is the Router MAC Address, as seen on your local, Ethernet network.

IP Address

This shows the Router IP Address, as it appears on your local, Ethernet network.

Subnet Mask

When the Router is using a Subnet Mask, it is shown here.
 
DHCP Server
The status of the Router's DHCP server function is displayed here.

Start IP Address

This shows the beginning of the range of IP addresses used by the DHCP Server.

End IP Address

This shows the end of the range of IP addresses used by the DHCP Server.
DHCP Client Table

DHCP Client Table

Clicking this button will open a screen showing you which PCs are utilizing the Router as a DHCP server. On the DHCP Client Table screen, you will see a list of DHCP clients (PCs and other network devices) with the following information: Client Names, IP Addresses, MAC Addresses, and the length of time before their assigned IP addresses expire.
 
ARP/RARP Table

ARP/RARP Table

Clicking this button will open a screen showing you which PCs are utilizing the Router as a ARP/RARP server. On the ARP/RARP Table screen, you will see a list of ARP/RARPs (PCs and other network devices) with the following information: IP Addresses and MAC Addresses.
