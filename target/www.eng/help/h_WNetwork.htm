Status Tab / Wireless


The Wireless screen on the Status Tab displays the status of your Wireless-B and/or Wireless-G networks.

Click the Refresh button to update the data displayed on screen.

Wireless Firmware Version
This is the firmware version of the Wireless module. This may be a different version number to the Router Firmware.

MAC Address

This is the Router MAC Address, as seen on your local wireless network.

Mode

As selected from the Wireless tab, this displays the Router Wireless networking mode.

SSID

As entered on the Wireless tab, this displays the wireless network name or SSID of your Wireless network.

DHCP Server

The status of the Gateway's DHCP server function is displayed here.

Channel

As entered on the Wireless tab, this displays the channel which your wireless network is using.

Encryption Function

This displays the data encryption status for your Wireless network. This is set on the Wireless Security screen.
 

Wireless Clients Connected

Click this button to see a list of the Wireless clients currently using the Wireless interface of this Router.
