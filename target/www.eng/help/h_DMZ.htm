Application and Gaming Tab / DMZ


The DMZ feature allows one network user to be exposed to the Internet for use of a special-purpose service such as Internet gaming or video conferencing. DMZ hosting forwards all the ports at the same time to one PC. The Port Range Forwarding feature is more secure because it only opens the ports you want to have opened, while DMZ hosting opens all the ports of one computer, exposing the computer to the Internet. Any PC whose port is being forwarded must have its DHCP client function disabled and should have a new static IP address assigned to it because its IP address may change when using the DHCP function.

DMZ

DMZ Hosting

Click the Enable radio button to enable the DMZ function as required. Because the DMZ PC is exposed to greater risk, the DMZ function should only be enabled when required.

Enter the IP address of the computer you want to expose.
